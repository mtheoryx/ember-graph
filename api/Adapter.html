Adapter Class

Defined in: src/adapter/adapter.js:5

An interface for an adapter. And adapter is used to communicate with the server. The adapter is never called directly, its methods are called by the store to perform its operations.

The adapter should return normalized JSON from its operations. Details about normalized JSON can be found in the pushPayload documentation.
